🚀 API ENDPOINTS
═══════════════════════════════════════════════════════════════

AUTHENTICATION
  POST   /api/auth/register      - Create new user
  POST   /api/auth/login         - Login user (get JWT token)

WORKOUTS
  GET    /api/workouts           - Get all workouts (authenticated)
  POST   /api/workouts           - Upload new workout (auth + file)
  GET    /api/workouts/:id       - Get specific workout
  DELETE /api/workouts/:id       - Delete workout (owner only)

STATISTICS
  GET    /api/users/:id/stats    - Get workout count
  GET    /api/users/:id/streaks  - Get current/longest streaks

LIKES (implied from schema)
  POST   /api/workouts/:id/like  - Like a workout
  DELETE /api/workouts/:id/like  - Unlike a workout

COMMENTS (implied from schema)
  POST   /api/workouts/:id/comments    - Add comment
  GET    /api/workouts/:id/comments    - Get comments
  DELETE /api/comments/:id              - Delete comment

🔧 KEY TECHNOLOGIES
═══════════════════════════════════════════════════════════════

Backend:
  • Node.js + Express (TypeScript)
  • SQLite database with foreign keys
  • JWT (jsonwebtoken) for authentication
  • bcryptjs for password hashing
  • Multer for file uploads
  • Cloudinary (optional) for cloud storage
  • CORS enabled

Frontend:
  • React 18 (TypeScript)
  • React Router DOM v6
  • Axios for API calls
  • Tailwind CSS for styling
  • Context API for state management
  • date-fns for date formatting
  • lucide-react for icons
